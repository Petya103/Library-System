# Library-System

1.Описание

Проектът представлява малка библиотечна система, реализирана на C++, която моделира книги, автори, читатели, заеми и библиотека.

Той демонстрира:

-обектно-ориентирано програмиране;

-конструктори/деструктори и Rule of 3/5;

-статични членове;

-капсулация, const-коректност и валидиране на данни;

-работа с композиция и колекции (std::vector).

2.Структура на проекта:

3.Компилация и изпълнение

Отворете терминал в директорията LibrarySystem/ и изпълнете:

g++ main.cpp -o output/main.exe


След това стартирайме програмата:

./output/main.exe

4.Кратко описание на класовете

Author

Представя автор с име и година на раждане.
Валидира се година (напр. 1850–2025).
to_string() връща представяне като текст.

Book

Съдържа заглавие, автор, година, цена и ISBN.
Брои живи книги чрез static int totalBooks.
Има конструктори, Rule 3 of 5.
Сетъри с валидиране (напр. цена ≥ 0).

Member

Моделира читател — име, ID и година на присъединяване.
Валидация за празно ID.

Loan

Представя заем на книга.
Съхранява ISBN, читател, начало, край, статус „върнато“.
Метод isOverdue() за проверка на просрочие.
Валидация, че dueDate >= startDate.

Library

Управлява колекции от книги, членове и заеми.
Добавяне на книги/читатели.
Заемане и връщане на книга.
Търсене по автор.
Проверка за наличност.
Съставен to_string() с обща статистика.
